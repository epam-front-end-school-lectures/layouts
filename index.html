z-index notes

  • Element with non-static positioning always will be above static element.
  • Element with positive z-index always will be above element without it.
  • On the same level the last element will be the highest one.
  • Lower amount of z-index elements is better.
  • In most cases you actually do not need z-index.

cheatsheet

The root element(viewport)
Positioned elements with negative z-index
Block-Level elements (non-positioned, non-floated)
Floated elements (non-positioned)
Inline elements (in-flow, non-positioned)
Positioned elements with z-index: 0; or z-index: auto;
Positioned elements with positive z-index
